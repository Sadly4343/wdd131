Color Schema

The color scheme that I will use is going to be green and red as they are complementary and provide a great color to match the fish and often times resembles fisherman colors. I will use red in the background and use green in the paragraphs and background colors with the header and background being red so create a complementary color scheme with the lettering staying as black for now.

Typography

The choice I made was Exo 2 and this is the current typography for this site because I find it to fit the website I am developing and creates an easier to read and more likeable site that provides the user with a better understanding and connection to the what is being presented.
